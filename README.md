# 坦克大战 🎮 (Tank Battle)

经典街机风格坦克大战游戏，使用纯 HTML + Canvas + JavaScript 实现。

## 游戏玩法

- **目标**：保护基地（底部金色鹰标），消灭所有敌方坦克
- **控制**：
  - `↑ ↓ ← →` 移动坦克
  - `Space` 射击
  - `P` 暂停/继续

## 游戏特性

- 🧱 **砖墙** — 可被子弹摧毁
- ⛓ **钢墙** — 需要强化子弹才能摧毁
- 🦅 **基地** — 被摧毁则游戏结束
- 🤖 **AI 敌人** — 自动追踪并攻击玩家
- ⭐ **道具系统** — 拾取提升能力
- 🌊 **波次系统** — 每波敌人逐渐变强

## 如何运行

直接用浏览器打开 `tank-battle.html` 即可游玩。

## 技术栈

纯前端，无依赖：
- HTML5 Canvas
- CSS3
- Vanilla JavaScript
